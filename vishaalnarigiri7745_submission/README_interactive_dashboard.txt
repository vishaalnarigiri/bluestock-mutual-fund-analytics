Bluestock Capstone Mutual Fund Analytics Dashboard:
The interactive dashboard is implemented using Streamlit instead of Power BI.
Source code can be found in Source_Code/dashboard/app.py.
To run the interactive dashboard, install requirements and execute:
    streamlit run Source_Code/dashboard/app.py
PDF export is available at Documentation/Dashboard.pdf.
Page screenshots are located in Documentation/charts/Page*.png.
